Área de uma Sala

Enunciado

Faça o algoritmo (diagrama de blocos e português estruturado) para calcular a área (AREA) de uma sala, sendo que os comprimentos das paredes (L e C) são fornecidos pelo usuário. Apresente a área depois de calculada.

Fluxograma
Pseudocódigo 
ALGORITMO CalcularArea
VAR
  L, C, AREAIgor: REAL
INICIO
  ESCREVA("Digite o comprimento L: ")
  LEIA(L)
  ESCREVA("Digite o comprimento C: ")
  LEIA(C)
  AREAIgor <- L * C
  ESCREVA("Área da sala: ", AREAIgor)
FIM
Código C++ 
#include <iostream>
using namespace std;

int main() {
    float L, C, AREAIgor;

    cout << "Digite o comprimento L: ";
    cin >> L;
    cout << "Digite o comprimento C: ";
    cin >> C;

    AREAIgor = L * C;

    cout << "Area da sala: " << AREAIgor << endl;
    return 0;
}

Porcentagem de Desconto

Enunciado

Faça o algoritmo (diagrama de blocos e português estruturado) para calcular qual foi a porcentagem de desconto dada em um determinado produto (DESC), sabendo-se o preço original do produto (PRECO) e o preço que foi cobrado por ele depois do desconto (PRECOF).

Fluxograma
Pseudocódigo 
ALGORITMO CalcularPorcentagemDesconto
VAR
  PRECO, PRECOF, DESCIgor: REAL
INICIO
  ESCREVA("Digite o preço original: ")
  LEIA(PRECO)
  ESCREVA("Digite o preço final: ")

  ESCREVA("Porcentagem de desconto: ", DESCIgor, "%")
FIM
Código C++ 
#include <iostream>
using namespace std;

int main() {
    float PRECO, PRECOF, DESCIgor;

    cout << "Digite o preco original: ";
    cin >> PRECO;
    cout << "Digite o preco final: ";
    cin >> PRECOF;

    DESCIgor = (PRECO - PRECOF) * 100 / PRECO;

    cout << "Porcentagem de desconto: "
         << DESCIgor << "%" << endl;
    return 0;
}

Valor do Estacionamento

Enunciado

Faça o algoritmo (diagrama de blocos e português estruturado) para calcular o valor a ser pago pelo período de estacionamento do automóvel (PAG). O usuário entra com os seguintes dados: hora (HE) e minuto (ME) de entrada, hora (HS) e minuto (MS) de saída.

Sabe-se que este estacionamento cobra R$ 4,00, mas calcula as frações de hora também. Por exemplo, se a pessoa ficar 1 hora e quinze minutos, pagará R$ 5,00 (R$ 4,00 pela hora e R$ 1,00 pelos quinze minutos).

Fluxograma
Pseudocódigo 
ALGORITMO CalcularEstacionamento
VAR
  HE, ME, HS, MS: INTEIRO
  TEMPO_MIN, PAGIgor: REAL
INICIO
  ESCREVA("Hora de entrada: ")
  LEIA(HE)
  ESCREVA("Minuto de entrada: ")
  LEIA(ME)
  ESCREVA("Hora de saída: ")
  LEIA(HS)
  ESCREVA("Minuto de saída: ")
  LEIA(MS)
  TEMPO_MIN <- (HS * 60 + MS) - (HE * 60 + ME)
  PAGIgor <- TEMPO_MIN * 4 / 60
  ESCREVA("Valor a pagar: R$ ", PAGIgor)
FIM
Código C++ 
#include <iostream>
using namespace std;

int main() {
    int HE, ME, HS, MS;
    float TEMPO_MIN, PAGIgor;

    cout << "Hora de entrada: ";   cin >> HE;
    cout << "Minuto de entrada: "; cin >> ME;
    cout << "Hora de saida: ";     cin >> HS;
    cout << "Minuto de saida: ";   cin >> MS;

    TEMPO_MIN = (HS * 60 + MS) - (HE * 60 + ME);
    PAGIgor = TEMPO_MIN * 4 / 60;

    cout << "Valor a pagar: R$ " << PAGIgor << endl;
    return 0;
}

Troca de Valores entre Variáveis

Enunciado

Faça o algoritmo (diagrama de blocos e português estruturado) que receba dois valores inteiros nas variáveis A e B. O programa deve trocar os valores entre as variáveis (ou seja, ao término do programa a variável A deve ter o valor inicial de B e vice-versa). Apresentar as duas variáveis ao final.

Fluxograma
Pseudocódigo 
ALGORITMO TrocarValores
VAR
  A, B, AUXIgor: INTEIRO
INICIO
  ESCREVA("Digite o valor de A: ")
  LEIA(A)
  ESCREVA("Digite o valor de B: ")
  LEIA(B)
  AUXIgor <- A
  A <- B
  B <- AUXIgor
  ESCREVA("Novo valor de A: ", A)
  ESCREVA("Novo valor de B: ", B)
FIM
Código C++ 
#include <iostream>
using namespace std;

int main() {
    int A, B, AUXIgor;

    cout << "Digite o valor de A: ";
    cin >> A;
    cout << "Digite o valor de B: ";
    cin >> B;

    AUXIgor = A;
    A = B;
    B = AUXIgor;

    cout << "Novo valor de A: " << A << endl;
    cout << "Novo valor de B: " << B << endl;
    return 0;
}

Quantidade de Tijolos

Enunciado

Faça o algoritmo (diagrama de blocos e português estruturado) que calcule a quantidade de tijolos iguais (QTD) necessários para construir uma determinada parede. São dados de entrada do programa: dimensões do tijolo (comprimento – CT e largura – LT) e dimensões da parede a ser construída (comprimento – CP e largura – LP).

Fluxograma
Pseudocódigo 
ALGORITMO CalcularTijolos
VAR
  CT, LT, CP, LP, QTDIgor: REAL
INICIO
  ESCREVA("Comprimento do tijolo (CT): ")
  LEIA(CT)
  ESCREVA("Largura do tijolo (LT): ")
  LEIA(LT)
  ESCREVA("Comprimento da parede (CP): ")
  LEIA(CP)
  ESCREVA("Largura da parede (LP): ")
  LEIA(LP)
  QTDIgor <- (CP * LP) / (CT * LT)
  ESCREVA("Quantidade de tijolos: ", QTDIgor)
FIM
Código C++ 
#include <iostream>
using namespace std;

int main() {
    float CT, LT, CP, LP, QTDIgor;

    cout << "Comprimento do tijolo (CT): ";
    cin >> CT;
    cout << "Largura do tijolo (LT): ";
    cin >> LT;
    cout << "Comprimento da parede (CP): ";
    cin >> CP;
    cout << "Largura da parede (LP): ";
    cin >> LP;

    QTDIgor = (CP * LP) / (CT * LT);

    cout << "Qtd de tijolos: " << QTDIgor << endl;
    return 0;
}
